# CSI4900_Honours_Project
Multi-turn LLM safety evaluation

## Notebooks
- **CSI4900_Honours_Project_Automatic.ipynb**: Automatic judge evaluation
- **CSI4900_Honours_Project_Manual.ipynb**: Manual analysis / inspection

## How to run
Open the notebooks in Jupyter/Colab. Install required packages as needed (e.g., datasets, transformers, evaluate, pandas). Also, in the beggining of each file you need to add Hugging_Face access token.
